{\itshape Tato semestrální práce patří do kategorie {\bfseries{interaktivních aplikací}}. Vaším cílem je vytvořit aplikaci, kterou bude uživatel interaktivně ovládat (např. pomocí příkazů na standardním vstupu). Případné parametry aplikace se nesnažte definovat přímo v kódu (a to ani pomocí konstant). Umístěte je do konfiguračních souborů (jednoho nebo více), které váš program bude načítat.}

   

Vaším úkolem je vytvořit kalkulačku pro výpočty s neomezenou přesností. Jednotlivá čísla a výsledky operací je možné ukládat do proměnných a tyto proměnné následně využívat v dalších výpočtech.

   

Kalkulačka musí implementovat následující funkcionality\+:

   
\begin{DoxyEnumerate}
\item 

Práce s celými i desetinnými čísly (především načtení a vypsání proměnných).


\item 

Základní operace\+: sčítání, odčítání, násobení.


\item 

Pro celá čísla implementujte navíc celočíselné dělení a zbytek po dělení.


\item 

Funkce pro zaokrouhlení, faktoriál a umocňování (exponent je celé nezáporné číslo).


\item 

Stav kalkulačky (tedy její proměnné) je možné uložit do souboru v textově čitelné formě. Stejně tak je potom možné tento stav ze souboru obnovit.


\item 

Uložení v rámci proměnných (a souboru) musí být paměťově efektivní. Při ukládání proměnné do paměti se musí určit typ daného čísla. Rozlišujte alespoň řídká čísla (obsahují především nulové hodnoty, př. 10$^\wedge$10) a hustá čísla (opak řídkých, př. 100! / 10$^\wedge$20) a podle toho určete efektivní reprezentaci.


\end{DoxyEnumerate}  

Dále aplikace musí zpracovávat výrazy dle jednoho z následujících principů\+:

   
\begin{DoxyEnumerate}
\item 

Základní operace (alespoň sčítání, odčítání a násobení) musí jít vnořovat pomocí závorek (př. {\ttfamily (3 + X) $\ast$ Y}), závorky mohou být libovolně vnořené. Funkce a složitější operace mohou fungovat formou individuálních příkazů bez dalšího skládání (př. {\ttfamily A = ROUND X -\/5}).


\item 

Pro zapisování výrazů využijte prefixový (nebo postfixový) zápis, mají tu výhodu, že nevyžadují závorky a snáze se zpracovávají. V tomto případě však veškeré operace a funkce musí jít zapsat v jednom výrazu (př. {\ttfamily A = ROUND MUL Y ADD 3 X -\/5}).


\end{DoxyEnumerate}

Další informace\+:

   
\begin{DoxyItemize}
\item 

Pro parsování výrazu se může hodit Shunting-\/\+Yard algoritmus (\href{http://en.wikipedia.org/wiki/Shunting-yard_algorithm}{\texttt{ http\+://en.\+wikipedia.\+org/wiki/\+Shunting-\/yard\+\_\+algorithm}}).


\item 

Pro uložení do souboru můžete využít i binární formát, v takovém případě dejte uživateli na výběr.


\item 

Nesnažte se ukládat čísla do textového řetězce, taková reprezentace je velice neefektivní.


\end{DoxyItemize}
